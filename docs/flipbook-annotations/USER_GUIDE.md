## Viewing Documents

### Accessing Documents

Documents can be accessed through:
- **Dashboard**: Your uploaded documents
- **Shared Links**: Documents shared with you
- **My Jstudyroom**: Purchased content (for members)

### Viewer Features

The flipbook viewer provides:
- **3D Page Turning**: Realistic page flip animations
- **Zoom Controls**: Zoom from 50% to 300%
- **Fullscreen Mode**: Immersive reading experience
- **Responsive Design**: Optimized for desktop, tablet, and mobile

## Navigation Controls

### Mouse/Trackpad Navigation

- **Click Left Edge**: Go to previous page
- **Click Right Edge**: Go to next page
- **Drag Pages**: Flip pages by dragging
- **Scroll Wheel**: Navigate through pages

### Keyboard Shortcuts

- **Arrow Left/Right**: Navigate pages
- **Home**: Go to first page
- **End**: Go to last page
- **Escape**: Exit fullscreen mode

### Touch Gestures (Mobile/Tablet)

- **Swipe Left**: Next page
- **Swipe Right**: Previous page
- **Pinch**: Zoom in/out
- **Double Tap**: Toggle fullscreen

### Control Buttons

- **Previous/Next Buttons**: Navigate pages
- **Page Counter**: Shows current page (click to jump to page)
- **Zoom In/Out**: Adjust viewing size
- **Fullscreen Toggle**: Enter/exit fullscreen mode

## Creating Annotations

### Who Can Create Annotations?

Only **Platform Users** (document owners) can create annotations. Members and readers can view annotations but cannot create them.

### Adding Audio Annotations

1. **Select Text**: Highlight the text you want to annotate
2. **Click "Add Audio"**: From the annotation toolbar
3. **Choose Source**:
   - **Upload File**: MP3, WAV (max 100MB)
   - **External URL**: YouTube, SoundCloud, or direct audio links
4. **Add Description**: Optional context for the annotation
5. **Click "Save"**: Annotation is created

### Adding Video Annotations

1. **Select Text**: Highlight the text you want to annotate
2. **Click "Add Video"**: From the annotation toolbar
3. **Choose Source**:
   - **Upload File**: MP4, WEBM (max 100MB)
   - **External URL**: YouTube, Vimeo, or direct video links
4. **Add Description**: Optional context for the annotation
5. **Click "Save"**: Annotation is created

## Managing Annotations

### Viewing Annotations

- **Click Marker**: Opens media player modal
- **Playback Controls**: Play, pause, volume, progress
- **Watermark**: User-specific watermark prevents unauthorized sharing

### Editing Annotations (Owner Only)

1. Click the annotation marker
2. Click "Edit" button
3. Update description or media source
4. Click "Save"

### Deleting Annotations (Owner Only)

1. Click the annotation marker
2. Click "Delete" button
3. Confirm deletion

### Annotation Visibility

- **Public**: Visible to all viewers (default)
- **Private**: Only visible to annotation creator

## Troubleshooting

### Common Issues

#### Pages Not Loading

**Problem**: Pages appear blank or don't load

**Solutions**:
- Check your internet connection
- Refresh the page (F5)
- Clear browser cache
- Try a different browser

#### Annotations Not Appearing

**Problem**: Annotation markers don't show up

**Solutions**:
- Ensure you're on the correct page
- Check if annotations are set to private
- Refresh the page
- Verify you have permission to view annotations

#### Media Won't Play

**Problem**: Audio/video doesn't play when clicking marker

**Solutions**:
- Check your internet connection
- Ensure browser supports media format
- Try disabling browser extensions
- Check if external URL is still valid

#### Upload Fails

**Problem**: Media file upload doesn't complete

**Solutions**:
- Check file size (max 100MB)
- Verify file format (MP3, WAV, MP4, WEBM)
- Check internet connection stability
- Try uploading a smaller file

#### Slow Performance

**Problem**: Flipbook is laggy or slow

**Solutions**:
- Close other browser tabs
- Disable browser extensions
- Use a modern browser (Chrome, Firefox, Safari, Edge)
- Check device performance
- Reduce zoom level

### Browser Compatibility

**Recommended Browsers**:
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

**Mobile Browsers**:
- iOS Safari 14+
- Chrome Mobile 90+
- Samsung Internet 14+

### DRM Protection

Documents are protected with:
- **Watermarks**: User-specific watermarks on all pages
- **Download Prevention**: Right-click and keyboard shortcuts disabled
- **Screenshot Detection**: Alerts when screenshots are attempted
- **Secure Streaming**: Media files use temporary signed URLs

### Getting Help

If you continue to experience issues:

1. **Check System Status**: Verify the platform is operational
2. **Contact Support**: Provide error details and screenshots
3. **Report Bugs**: Use the error reporting feature
4. **Documentation**: Review technical documentation for advanced features
