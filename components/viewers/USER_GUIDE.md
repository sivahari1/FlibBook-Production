# SimpleDocumentViewer User Guide

## Getting Started

The SimpleDocumentViewer provides a modern, full-screen document viewing experience with intuitive navigation controls. This guide covers all the features and how to use them effectively.

## Interface Overview

### Toolbar

The toolbar at the top of the viewer contains all navigation and control options:

- **Document Title**: Shows the name of the current document
- **Close Button** (X): Closes the viewer and returns to the previous page
- **Page Navigation**: Arrow buttons and page input for moving between pages
- **Zoom Controls**: Buttons to zoom in and out, with current zoom percentage
- **View Mode Toggle**: Switch between continuous scroll and paged view

### Document Area

The main area displays your document content:
- **Continuous Mode**: Pages flow vertically with smooth scrolling
- **Paged Mode**: Shows one page at a time with discrete navigation
- **Watermark**: May appear as an overlay (if enabled by document owner)

## Navigation Methods

### 1. Mouse Navigation

#### Page Navigation
- **Previous/Next Arrows**: Click the arrow buttons in the toolbar
- **Page Input**: Click the page number field and type a specific page number
- **Scrolling** (Continuous Mode): Use mouse wheel to scroll through pages

#### Zoom Control
- **Zoom Buttons**: Click + or - buttons in the toolbar
- **Ctrl + Scroll**: Hold Ctrl (or Cmd on Mac) and scroll to zoom in/out

### 2. Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `↓` or `Page Down` | Go to next page |
| `↑` or `Page Up` | Go to previous page |
| `Home` | Go to first page |
| `End` | Go to last page |
| `Ctrl/Cmd + +` | Zoom in |
| `Ctrl/Cmd + -` | Zoom out |

### 3. Touch Gestures (Mobile)

| Gesture | Action |
|---------|--------|
| Swipe Left | Next page |
| Swipe Right | Previous page |
| Pinch Out | Zoom in |
| Pinch In | Zoom out |

## View Modes

### Continuous Scroll Mode

**Best for**: Reading through documents sequentially, like books or reports.

**Features**:
- Pages flow vertically in a continuous stream
- Smooth scrolling between pages
- Page indicator updates automatically as you scroll
- Ideal for reading long documents

**How to use**:
1. Scroll with mouse wheel or trackpad
2. Use keyboard arrows for page-by-page navigation
3. Current page indicator shows which page is most visible

### Paged Mode

**Best for**: Reviewing specific pages, presentations, or when you need precise page control.

**Features**:
- Shows one page at a time
- Discrete navigation between pages
- Page is centered and optimized for viewing
- Ideal for presentations or detailed page review

**How to use**:
1. Use arrow buttons or keyboard shortcuts to navigate
2. Enter specific page numbers in the page input field
3. Each page is displayed individually and centered

### Switching Between Modes

Click the view mode toggle button in the toolbar:
- **Book icon**: Currently in continuous mode (click to switch to paged)
- **Scroll icon**: Currently in paged mode (click to switch to continuous)

**Note**: Your current page position is preserved when switching modes.

## Zoom Controls

### Zoom Levels
- **Minimum**: 50% (0.5x)
- **Default**: 100% (1.0x)
- **Maximum**: 300% (3.0x)

### Zoom Methods
1. **Toolbar Buttons**: Click + or - buttons
2. **Keyboard**: Ctrl/Cmd + Plus/Minus
3. **Mouse**: Ctrl/Cmd + Scroll wheel
4. **Touch**: Pinch gestures on mobile devices

### Zoom Behavior
- Zoom increments/decrements by 25% each step
- Current page remains visible when zooming
- Zoom level is displayed as a percentage in the toolbar
- Zoom preference is saved for future sessions

## Page Navigation

### Direct Page Access
1. Click on the page number in the toolbar
2. Type the desired page number
3. Press Enter or click elsewhere to navigate

### Sequential Navigation
- **Next Page**: Arrow buttons, keyboard shortcuts, or swipe gestures
- **Previous Page**: Arrow buttons, keyboard shortcuts, or swipe gestures
- **First/Last Page**: Home/End keys

### Page Indicator
The page indicator shows "Page X of Y" format:
- Updates automatically as you navigate
- Reflects the currently visible page in continuous mode
- Shows the displayed page in paged mode

## Troubleshooting

### Common Issues

#### Pages Not Loading
**Symptoms**: Blank pages or loading indicators
**Solutions**:
1. Check your internet connection
2. Refresh the page
3. Try a different browser
4. Contact support if problem persists

#### Slow Performance
**Symptoms**: Laggy scrolling or navigation
**Solutions**:
1. Close other browser tabs
2. Reduce zoom level
3. Switch to paged mode for large documents
4. Clear browser cache

#### Keyboard Shortcuts Not Working
**Symptoms**: Keys don't trigger navigation
**Solutions**:
1. Click inside the viewer area to focus it
2. Check if other browser extensions are interfering
3. Try using mouse navigation instead

#### Mobile Touch Issues
**Symptoms**: Gestures not recognized
**Solutions**:
1. Ensure you're using supported gestures
2. Try single-finger swipes for navigation
3. Use two fingers for pinch-to-zoom
4. Refresh the page if gestures stop working

### Error Messages

#### "No pages available to display"
- The document has no content or failed to load
- Try refreshing or contact the document owner

#### "Failed to load page X"
- Individual page failed to load
- Click the retry button on the failed page
- Check your internet connection

#### "Invalid page data"
- Document structure is corrupted
- Contact support or try a different document

## Tips and Best Practices

### For Reading
1. **Use continuous mode** for sequential reading
2. **Adjust zoom** to comfortable reading size
3. **Use keyboard shortcuts** for efficient navigation
4. **Bookmark important pages** by noting page numbers

### For Presentations
1. **Use paged mode** for slide-by-slide control
2. **Practice navigation** before presenting
3. **Use keyboard shortcuts** for smooth transitions
4. **Test on the presentation device** beforehand

### For Mobile
1. **Use landscape orientation** for better page visibility
2. **Pinch to zoom** for detailed viewing
3. **Swipe gestures** are faster than button taps
4. **Close other apps** for better performance

### For Accessibility
1. **Use keyboard navigation** if mouse is difficult
2. **Increase zoom** for better text visibility
3. **Enable screen reader** for audio feedback
4. **Use high contrast mode** if available in your browser

## Privacy and Security

### Watermarks
- Some documents may display watermarks
- Watermarks don't interfere with navigation
- They help protect document copyright

### Screenshot Prevention
- Some documents may prevent screenshots
- This is a security feature set by the document owner
- Use print functionality if available and permitted

### Data Storage
- Viewing preferences are stored locally in your browser
- No personal data is transmitted to external servers
- Clear browser data to reset preferences

## Support

If you encounter issues not covered in this guide:

1. **Check browser compatibility**: Ensure you're using a supported browser
2. **Update your browser**: Use the latest version for best performance
3. **Disable extensions**: Some browser extensions may interfere
4. **Contact support**: Provide details about your issue and browser version
